この画像は文書ページのスクリーンショット（仕様書、報告書、議事録など）です。
画像内に**図表やチャートは含まれていません**。テキスト情報のみで構成されています。

以下のルールに従い、テキスト内容を **Markdown 形式で完全に転写** してください。

## 転写ルール

1. **見出し**: 文書のセクション見出しは `#` / `##` / `###` で表現
2. **段落**: 本文の段落はそのまま Markdown の段落として記述（1 文字も省略しない）
3. **箇条書き**: `- ` で表現。番号付きの場合は `1. ` で
4. **表**: Markdown table で転写。ヘッダー・全セルの値を正確に
5. **数値・記号**: 単位、不等号（≤, ≥）、パーセント、時間表記をそのまま保持
6. **ヘッダー・フッター**: ページ番号、日付、文書名、機密レベルなども漏れなく記述

## 禁止事項

- テキストの要約・言い換え・省略
- 画像に書かれていない情報の補完
- 見えにくい部分の推測（読めない場合は `[判読不能]` と記す）

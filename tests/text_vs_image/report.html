Text vs Image Fidelity Report (全自動 LLM 評価版)

画像を LLM に渡す 2 通りの経路（specialized 記述を経由 vs 画像を直接渡す）でどの程度の情報損失が起きるか、またプロンプト工夫で記述の質をどれだけ改善できるかを LLM (Gemini) を使って完全自動で評価。Copilot Chat の代替として同じ Gemini モデルを使用しているため、同一モデル内の自己参照バイアスがあり得ることに留意。
